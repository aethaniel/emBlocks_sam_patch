  • ACEN: Analog Comparator ENable
    ValueNameDescription
    0DISAnalog Comparator Disabled.
    1ENAnalog Comparator Enabled.
  • EDGETYP: EDGE TYPe
    ValueNameDescription
    0x0RISINGonly rising edge of comparator output
    0x1FALLINGfalling edge of comparator output
    0x2ANYany edge of comparator output
  • INV: INVert comparator output
    ValueNameDescription
    0DISAnalog Comparator output is directly processed.
    1ENAnalog Comparator output is inverted prior to being processed.
  • SELFS: SELection of Fault Source
    ValueNameDescription
    0CFthe CF flag is used to drive the FAULT output.
    1OUTPUTthe output of the Analog Comparator flag is used to drive the FAULT output.
  • FE: Fault Enable
    ValueNameDescription
    0DISthe FAULT output is tied to 0.
    1ENthe FAULT output is driven by the signal defined by SELFS.
